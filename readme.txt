Program přejmenuje soubory ve složce, kde se nachází.

Náhodný režim:
Přejmenuje všechny soubory náhodně.
Užitečné například pokud chceme na ploše používat background slideshow, a chceme, aby byly vybírány náhodně. Background slideshow vždy postupuje postupně tak, jak jsou soubory řazeny ve Windows Explorer, který sám řadit soubory náhodně neumí.
